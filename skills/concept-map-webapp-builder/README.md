# concept-map-webapp-builder

临时同步自 `lightcoloror/research-skills` 的关系图工作台 skill。

它不是保险经纪专用 skill，而是用于把概念、流程、代码知识、工具雷达、证据链和系统关系整理成可观察、可交互的关系图或本地 web 工作台。

## 适用场景

- 整理复杂概念、工作流程、工具组合或代码知识。
- 生成可拖拽、可缩放、可聚焦的关系图。
- 为 System map、Observable brain、代码图谱、证据图谱或工具雷达建立可检查的图数据和布局规则。

## 风险边界

- 不应把 AI 生成的图谱当成最终事实，需要保留证据、来源、验证状态和人工复核边界。
- 不应让前端直接打开任意本地路径或执行任意命令；如需本地控制，必须使用白名单 gateway。
- 不应把这个临时副本当成上游最新版。长期维护仍以 `research-skills` 中的原 skill 为准。
