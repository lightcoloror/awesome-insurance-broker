# Hooks and Headlines

## Purpose

The hook interrupts scrolling. The first line earns attention.

## Useful title families

- 干货式
- 对比式
- 故事式
- 热点式
- 悬念式
- 反问式

## Useful opening patterns

- 反常识判断
- 一个真实现场
- 一个问题
- 一个代价提醒

## First-line test

The first line should do at least one of:

- Create curiosity
- Create tension
- Create relevance
- Create emotional entry
